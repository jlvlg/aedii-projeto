◆ inorder

void(* inorder) (Tree root, void function(Tree))

Transverses the tree and calls a function as it goes.

Order of execution:

  1. Tries to access left node if possible
  2. When no longer possible, calls function on current node
  3. Tries to access right node if possible
  4. When no longer possible, resumes from previous node

May be used to obtain the values of a tree in order in O(n)

void(* postorder) (Tree root, void function(Tree))

Transverses the tree and calls a function as it goes.

Order of execution:

  1. Tries to access left node if possible
  2. When no longer possible, tries to access right node
  3. When no longer possible, calls function on current node
  4. Resumes from previous node

May be used if you need to access a node's children values before the node's own
